AltHealth Information System

This project was given to me as my final university assessment and was my first big project relative to everything else that I had done previously. We were given the opportunity to use any technologies to build it and thus began my journey with C#.Net and MS SQL Server.

The clients was a medical institution that sold medical supplies to its customers. There would be 3 user logins with different levels of access. The system would need to store patient and product data then process it to generate orders, invoices and reports for management. I also added an admin function to update password and update other values.

In order to run or build the app in visual studio you would need to add the following:

  • Source Code
  • Microsoft.SqlServer.SqlManagementObjects (NuGet Package)
  • Microsoft.ReportingServices.ReportViewerControl.Winforms (NuGet Package)
  • Microsoft SQL Server 2017 Express
  • Demo Database
  • App Login Details (Username: Super User & Password: admin)
